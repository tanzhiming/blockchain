# 构建第一个网络

构建第一个网络的场景提供了一个样例网络, 它由两个组织（每个组织维护两个peer节点）和一个"solo" 排序服务组成。

## 安装准备

首先应安装好样例、二进制和Docker镜像。然后切换到目录

```shell
cd fabric-samples/first-network
```

## 脚本方式构建网络

### 生成网络构件

```shell
./byfn.sh generate
```

### 启动网络

```shell
./byfn.sh up
```

### 停止网络

```
./byfn.sh down
```
